Key Features

Enterprise-grade fraud prevention built for WHMCS hosting companies.

16 Detection Engines

FraudRecord, AbuseIPDB, IPQualityScore, MaxMind GeoIP, StopForumSpam, SpamHaus, Turnstile, BIN/IIN, domain RDAP, device fingerprint, behavioral biometrics, velocity engine, Tor detection, Safe Browsing, global intel, bot pattern.

3D Threat Topology

Three.js globe with real-time fraud event arcs, country hotspots, live threat feed, 6 time ranges, and auto-refresh. Works in admin panel and as standalone fullscreen page.

REST API (4 Tiers)

Public API with Anonymous, Free, Basic ($19/mo), and Premium ($99/mo) tiers. Token bucket rate limiting, SHA-256 key hashing, IP whitelisting, usage tracking dashboard.

Bot Detection & Cleanup

Financial-data-based bot detection with 100% accuracy. Preview before purge. Deep purge mode. Orphan user cleanup for WHMCS 8.x. Intel harvested before deletion.

Device Fingerprinting

Canvas, WebGL, fonts, screen, audio, WebRTC fingerprinting. Passive collection on every page. Cross-account matching in duplicate detection.

Global Intel Hub

Hub-and-spoke cross-instance threat sharing. Auto-push via daily cron. Fast lookup (<200ms local, <500ms with hub). GDPR-compliant with purge controls.

Velocity Engine

5 rate limit dimensions: orders/IP, registrations/IP, failed payments, checkout attempts, BIN reuse. Configurable windows and thresholds.

Geo-Impossibility

Cross-correlates IP country, billing address, phone number, and BIN country. Detects impossible geographic combinations.

Behavioral Biometrics

Mouse entropy, keystroke cadence, form fill speed, paste detection. Distinguishes human vs bot behavior patterns.

Webhook Notifications

Slack, Discord, Teams, and generic webhook support. SSRF-hardened with HTTPS-only, DNS resolution, and internal IP blocking.

GDPR Compliance

Art. 17 right to erasure, Art. 20 data portability. Self-service removal form. Non-enumerable responses. Hub purge propagation.

Accessibility

Colorblind-friendly toggle on every page. Swaps green to blue with WCAG 2.1 compliant contrast ratios, underlined links, and focus rings.
